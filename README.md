# Fibonacci

Verifica se um numero gerado aleatoriamente pertence a série de *Fibonacci* gerada ate *350*.

> A série de Fibonacci é a seguinte: 0, 1, 1, 2, 3, 5, 8, 13, etc...

## Tópicos

Neste exercicio eu aprendi:

- Variáveis e métodos
- Operadores matemáticos
- Controle de fluxo
- Listas

## Recursos utilizados
​
- Java 8 +
